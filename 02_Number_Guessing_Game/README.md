# Number Guessing Game

A beginner Python game where the player has to guess a randomly generated number within a limited number of attempts.

## Features

-  Three difficulty levels
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–500)
-  Random number generation
-  Limited attempts based on difficulty
-  Hints:
  - Too High
  - Too Low
  - High, but nearby
  - Low, but nearby
-  Input validation using `try-except`
-  Exit the game anytime
-  Displays remaining attempts
-  Reveals the correct number if all attempts are exhausted

## Concepts Used

- Variables
- `if`, `elif`, `else`
- `for` loops
- `while` loops
- Nested loops
- `break` and `continue`
- `try-except`
- Functions from the `random` module
- User input validation

## How to Run

```bash
python3 number_guessing_game.py
```
